  - Cross-topic bridge detection is entity-window based and should be tightened once more historical data is available.
  - Forecast resolution currently supports deterministic target families (`thread_new_event_count`, `topic_event_growth`, `entity_recurrence`, `cross_topic_follow_on`) and should expand as richer signal types are added.
  - The forecast catalog now includes semantic event families (`official_statement_event`, `legal_action_event`, `partnership_or_product_event`, `infrastructure_milestone_event`). Calibration fix on 2026-03-15 eliminated the `0.80-0.95` overconfidence; remaining issue is the `0.60-0.80` bucket (12 forecasts, 0% hit rate) from momentum-boosted `about_to_break` threads.

  - Parallel CLI processes can contend on SQLite writes; schema initialization is now safer, but replay/generate runs should still be scheduled sequentially against the main DB.
